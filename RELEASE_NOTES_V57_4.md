# Release Notes V57.4 — Internal Legacy Cleanup

هذه نسخة تنظيف داخلي فقط فوق V57.3.

## ما تم
- تحديث النسخة والكاش إلى `57_4legacyclean`.
- الحفاظ على 8 ملفات JavaScript فقط.
- إزالة ملفات توثيق V57.3 القديمة من الحزمة.
- الإبقاء على ملفات توثيق V57.4 فقط.
- تنظيف تعليقات Legacy غير المستخدمة داخل `01-core-base.js` بدون حذف منطق تشغيل.
- تحديث Quality Gate ليعرض V57.4.
- عدم إضافة Features جديدة.
- عدم لمس Apps Script.
- عدم تغيير الحسابات أو الداتا أو شكل الاستخدام اليومي.

## سبب عدم حذف كود كبير
لا يزال داخل `01-core-base.js` و `02-business-legacy.js` بعض كود Legacy تشغيلي. لم يتم حذفه لأنه جزء من Boot/Fallback ومسارات قديمة قد يطلبها النظام أثناء التحميل. الحذف الكبير قد يسبب أخطاء تشغيل، لذلك تم تنظيف الآمن فقط.
