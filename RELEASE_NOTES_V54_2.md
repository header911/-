# Release Notes V54.2 Lite Repair

## سبب الإصدار
إصلاح أخطاء ظهرت في V54/V54.1:
- `Unexpected end of input` عند الضغط على أزرار التقارير.
- أزرار `كشف` و `فتح` داخل V53/V52 لا تعمل.
- أزرار أرشيف المستندات في V54 معرضة لنفس مشكلة الـ inline onclick.
- زيادة ملفات التوثيق في جذر GitHub.

## ما تم إصلاحه
- إزالة كل الـ inline onclick الديناميكي الذي يحتوي IDs من V52/V53/V54.
- استبدال أزرار التقارير بـ `data-*` attributes + event delegation.
- إصلاح أزرار:
  - كشف عميل.
  - كشف مصنع.
  - فتح أوردر.
  - تفاصيل شهر.
  - فتح/طباعة مستند.
  - تغيير حالة المستند.
- حذف ملفات التوثيق القديمة من الـ ZIP لتخفيف عدد ملفات الرفع.
- تحديث الكاش إلى `54_2litefix`.

## لم يتم تغييره
- لا يوجد تعديل Apps Script.
- لا حذف للبيانات.
- لا حذف لميزات V49.1/V50/V51/V52/V53/V54.
- لا إضافة ملفات JavaScript جديدة.
