# V55.3 Documents Stable Repair

هدف النسخة: إصلاح المستندات والطباعة بعد V55.2 بدون إضافة Features جديدة وبدون لمس Apps Script.

## إصلاحات أساسية
- إصلاح أزرار فتح/طباعة المستندات من صفحة التقارير بإضافة `openRecord` الناقصة.
- إزالة شارة `Draft` من المستندات المطبوعة/المفتوحة.
- إزالة سطر الملاحظة أسفل الفاتورة: "تم إصدار الفاتورة...".
- إزالة بلوك التوقيعات والـ footer من نسخة الطباعة حسب طلب المستخدم.
- تجهيز عنوان نافذة الطباعة باسم واضح للـ PDF مثل:
  - فاتورة محمد وحيد - HP-2026-INV-0001
  - عرض سعر محمد وحيد - HP-2026-QT-0001
  - كشف حساب محمد وحيد - HP-2026-CS-0001
- معالجة المستندات القديمة المحفوظة Sanitization وقت الفتح حتى لا تظهر العناصر المحذوفة.

## لم يتم تغييره
- لا يوجد تعديل Apps Script.
- لا يوجد تعديل على الداتا.
- لا يوجد إضافة ملفات JavaScript جديدة.
- عدد ملفات JavaScript كما هو 8 ملفات.
